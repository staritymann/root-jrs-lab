# root-jrs-lab

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态 HTML 资料的集合与演示用途。

## 目录说明

- `/pages`：存放各独立 HTML 页面的主目录
- `/assets`：页面所需的公共资源（如样式、脚本、图片等）
- `/docs`：相关文档与使用说明（可选）

## 页面归档说明

本仓库中的 HTML 页面均为独立、自包含的静态文件，可直接在浏览器中打开预览。页面内容涵盖多种类型，包括但不限于工具页面、信息展示页、交互式演示等。每个页面均无外部依赖，或仅引用 `/assets` 下的本地资源。

## 维护说明

- 新增页面：将 HTML 文件放入 `/pages` 目录，并在本文件底部更新页面列表。
- 资源更新：如需修改全局样式或脚本，请更新 `/assets` 中对应的文件。
- 删除页面：移除对应文件，并从列表中删除条目。
- 请确保每个页面在提交前经过基本测试，避免链接或资源错误。
